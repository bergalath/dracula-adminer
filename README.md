# Dracula for [Adminer](https://www.adminer.org/)

> A dark theme for [Adminer](https://www.adminer.org/).

![Screenshot](./screenshot.png)

## Install

Copy [adminer.css](./adminer.css) alongside Adminer PHP script to use design.
